# Customer Churn Analytics

## Project Overview
This project analyzes customer churn in a telecom dataset and builds machine learning models to predict whether a customer is likely to leave. In addition to prediction, the project focuses on business interpretation, model comparison, threshold tuning, and feature importance analysis.

## Goals
- understand churn patterns in customer data
- build predictive models for churn classification
- compare multiple models
- tune the best model
- identify the most influential churn drivers

## Dataset
This project uses the **Telco Customer Churn** dataset from Kaggle.

The raw dataset is not included in this repository.

## Workflow
1. Data cleaning
2. Exploratory Data Analysis
3. Preprocessing pipeline
4. Baseline Logistic Regression
5. Model comparison
6. Random Forest tuning
7. Threshold tuning
8. Feature importance analysis
